== Description ==

SKT Generic is a simple, flexible, multipurpose theme developed using block patterns and editable using Gutenberg editor. This theme comes with 5 premade block patterns and can be used for minimal, corporate, business, architect, interior, portfolio and other types of photography industry usage.
